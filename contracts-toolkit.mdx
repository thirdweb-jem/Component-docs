# Smart Contract Toolkit

[Contracts](https://thirdweb.com/contracts) is a complete toolkit for writing building and deploying smart contracts for any developer and any app. 

With secure and audited smart contract templates, developers have the choice to deploy reliable smart contracts in just a few clicks. For building custom smart contracts, the modular contract framework offers a flexible solution, with the option to add functionalities and amend capabilities at a later date. And for seasoned developers, the Solidity SDK offers the tools to write custom smart contracts. Finally, the smart contract deployment tool, and decentralized storage for data.

---

### Smart Contract toolkit features:

[Pre-Built Smart Contract Templates](https://portal.thirdweb.com/contracts/explore/overview) : Ready-to-deploy, pre-audited smart contract templates for deploying apps in a matter of clicks.

[Modular Contract Framework](https://portal.thirdweb.com/contracts/modular-contracts/overview) : Building blocks comprising core contracts and modules, helping developers build contracts based on the functionalities they need with the power to upgrade them later.

[Solidity SDK](https://portal.thirdweb.com/contracts/build/overview) : Tools for writing your own custom smart contracts from scratch
