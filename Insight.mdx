# Blockchain Indexer

[**thirdweb Insight**](https://thirdweb.com/insight) is a lightning-fast blockchain indexer that lets developers transform and query onchain data across **30+ chains**.  

Insight includes out-of-the-box APIs for **transactions**, **events**, **tokens**, and gives developers the power to define **custom API schemas** (blueprints) â€” without the need for ABIs, decoding, or RPCs.

---

## Use Cases for thirdweb Insight

thirdweb Insight can be used to fetch:

- Tokens (ERC20, ERC721, ERC1155) for a given wallet address  
- Historical purchasing data for onchain marketplaces  
- Total cost of all accounts created using ERC-4337  
- Token metadata (ERC20, ERC721, ERC1155)  
- Daily active wallets for your application or game  
- And much more  

[Learn more about thirdweb Insight in the technical docs](https://portal.thirdweb.com/insight)
